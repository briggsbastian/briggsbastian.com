UniFi runs the network; NixOS runs the bones. The lab is a mixed estate —
three Proxmox hypervisors carrying NixOS VMs, NixOS on the metal where it
matters, and exactly one machine that got grandfathered in. There is no
wiki for it; this page is the closest thing the lab has to up-to-date
documentation, which is itself an argument for the declarative parts.

## Base topology

The trusted network is where the fleet lives:

| Host             | Role                                                       |
| ---------------- | ---------------------------------------------------------- |
| `pve · media`    | Proxmox host — NixOS VMs for the media stack               |
| `pve · security` | Proxmox host — security tooling VM deployment              |
| `pve · netmgmt`  | Proxmox host — network management VMs                      |
| `gaming rig`     | NixOS desktop, GPU and all                                 |
| `game server`    | GNOME NixOS box for hosting game servers                   |
| `nas`            | Ubuntu 24.04 LTS — bulk storage, the non-NixOS holdout     |

Click any hypervisor in the map to expand the NixOS VMs it carries.
Segmentation is UniFi policy at the gateway: the IoT network holds two
smart TVs and a HomePod that can reach the internet but not the lan, and
visitors land on a guest network that gets them online and nowhere else.

## What it proves

- **Declarative where it counts.** The NixOS machines — VMs and metal —
  are rebuildable from configuration, not from memory. The undocumented
  parts of the lab are, not coincidentally, the parts that aren't NixOS
  yet.
- **Virtualization as a discipline.** Three hypervisors with deliberate
  roles — media, security, network management — instead of one box
  running everything that seemed like a good idea at 1 a.m.
- **Pragmatism over purity.** The NAS runs Ubuntu LTS because storage
  should be boring, and migrations should happen for reasons. It's on the
  list; it's not urgent.

The lab is also the dogfood pen for [NixRMM](/projects/nixrmm/): the
NixOS VMs across the Proxmox hosts are exactly the kind of fleet its
Phase 1 targets — and the game server is where
[Gesture](/projects/gesture/)'s dedicated builds will prove themselves.
